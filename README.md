# distributed-ecommerce

Mini e-commerce distribuído feito pra disciplina de FCCPD. Cinco
microsserviços escritos em Go que se comunicam via HTTPS por trás de
um API Gateway, com replicação do serviço de produtos, heartbeat de
detecção de falhas e um dashboard simples no navegador pra ver o
status e simular quedas.

A configuração, o fluxo de demo e as instruções de execução estão no
[`README_execucao.md`](./README_execucao.md). O relatório com as
cinco perguntas do enunciado respondidas está em `relatorio.pdf`.
