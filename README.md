# distributed-ecommerce

A distributed mini e-commerce built for the FCCPD coursework. Five microservices
written in Go, communicating over HTTPS through an API gateway, with strong-consistency
product replication, gateway-driven heartbeats, and a small browser dashboard for
status and outage simulation.

Setup, demo flow and configuration live in [`README_execution.md`](./README_execution.md).
The accompanying report (5 required questions answered) is in `report.pdf`.
